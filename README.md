# BIMVAA STUDIO — website

Static photography & film site for BIMVAA STUDIO (Sydney). No build step.

## Editing content
Photos, films and contact details are managed with **Pages CMS** at
https://app.pagescms.org — sign in with GitHub, open this repo, and edit.
Changes are committed here and auto-deploy via Cloudflare Pages.

- Portfolio photos: `content/portfolio.json`
- Films: `content/films.json`
- Site & contact: `content/site.json`
- Uploaded images: `media/`
- CMS config: `.pages.yml`

## Pages
`index.html` (home) · `about.html` · `book.html` (enquiry) · `journal/`
